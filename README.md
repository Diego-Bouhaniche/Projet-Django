# Projet-Django
https://openclassrooms.com/fr/courses/7172076-debutez-avec-le-framework-django

Tout au long de ce cours, nous vous guiderons pas à pas dans la création d'une application web Django amusante et entièrement fonctionnelle.

Tout d'abord, nous allons jeter les bases de notre application à l'aide de modèles, de vues et de gabarits. 
Ensuite, nous allons gérer les données de notre application, d'abord en utilisant le site d’administration de Django, puis en construisant une interface CRUD.
